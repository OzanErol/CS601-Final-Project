# Portfolio Website

This is a comprehensive portfolio website built as part of the MET CS601 Web Application Development course. It showcases skills in HTML5, CSS3, JavaScript, TypeScript, React, and web design principles.

## Project Overview

The website includes the following pages:
- **Home**: Introduction and welcome message
- **About**: Biography and interests
- **Skills**: Technologies learned in the course
- **Projects**: Showcase of course projects
- **Resume**: Professional resume
- **Contact**: Web-based contact form
- **Gallery**: Photo gallery

## Technologies Used

- **React** with TypeScript for component-based development
- **React Router** for client-side routing
- **CSS3** for responsive styling with Flexbox
- **HTML5** semantic elements
- **JavaScript/TypeScript** for interactivity and DOM manipulation

## Features

- Responsive design that works on mobile and desktop
- Navigation between pages
- Interactive contact form with validation
- Photo gallery with sample images
- Accessibility considerations (alt text, labels, etc.)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This project can be deployed to platforms like Netlify, GitHub Pages, or Azure Static Apps.
